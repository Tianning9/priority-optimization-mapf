# priority-optimization-mapp
Master’s thesis project on priority optimization for sequential multi-agent path planning (MAPP).

## Repository Structure

```
repository/
│
├── thesis/
│    └── thesis.pdf
│
├── code/
│   ├── main.py
│   └── other implementation files
│  
└── README.md
```
---

## Code

The implementation of the simulation and optimization algorithms is located in the `code` directory.
Detailed instructions for installing dependencies and running the experiments can be found in:
code/README.md

---

## Getting the Code

Clone the repository from GitHub:
git clone https://github.com/Tianning9/Priority-Optimization-MAPP.git
Alternatively, you can download the repository directly from the GitHub project page using **Download ZIP**.
After cloning the repository, navigate to the `code` directory to run the experiments:
cd code
python main.py
